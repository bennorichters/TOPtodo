TOPtodo Privacy Policy

We do not collect any personal data.

TOPtodo only communicates with a TOPdesk server using the TOPdesk API. The URL of this TOPdesk server is provided by the user of the TOPtodo mobile application. Please refer to your TOPdesk application manager for the privacy policy of the TOPdesk server you wish to use.

The TOPtodo mobile application stores your TOPdesk login credentials securily on your mobile device.
